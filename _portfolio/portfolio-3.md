paper_abstract: >-
  This work presents a LiDAR-based quadrotor inspection system for cluttered, unstructured, and GNSS-denied environments. It combines a human-in-the-loop setup phase, in which even inexperienced pilots can map the site and mark inspection targets with autonomous obstacle avoidance, with a fully autonomous execution phase. The recorded targets are reordered through traveling-salesman optimization using collision-free path costs, enabling efficient and repeatable data collection. Field tests across slopes, landslides, agricultural fields, factories, and forests demonstrate reliable deployment, reducing trajectory length by up to 40% and flight time by up to 57% compared with human-guided inspection.
